Introduction

Pourquoi Wordpress ?

Wordpress est un CMS, Content Manager System, un système de gestion de contenu en français.

L'objectif du CMS est de faciliter la création d'un site web, sa maintenance et son administration. C'est un outil développé par des informaticiens destiné aux non-informaticiens pour qu'ils puissent administrer un site web.

Wordpress va donc nous permettre de réaliser tout ce qu'un administrateur de site web fait au quotidien de façon simple : créer un site web, personnaliser son visuel, créer du contenu sur ce site web, le mettre à jour, etc.

Voici quelques sites web professionnels dévéloppés avec Wordpress en guise d'exemple.

Objectif du tutoriel

L'objectif du tutoriel est d'apprendre à maîtriser les bases de Wordpress :

  • Installer wordpress ;

  • Créer son site web ;

  • Poster des articles ;

  • Créer des pages web ;

  • Créer un menu de navigation dans ces pages ;

  • Personnaliser le contenu des pages et articles ;

  • Personnaliser l'identité visuelle du site web.

Dans la suite du tutoriel, nous créerons nous un site web pour l'IUT GEA, mais vous pouvez prendre n'importe quel exemple, peu importe.
