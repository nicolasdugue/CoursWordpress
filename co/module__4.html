Administrer son site web

Le site web

Voici donc votre site web Wordpress créé comme ci-dessous. Au centre, vos articles de blogs ou "posts" en anglais. A droite, la sidebar, la colonne latérale du site web.

Nous allons voir comment personnaliser ce site web, son aspect visuel, son contenu et ses menus.

Méthode : Connexion

Pour pouvoir personnaliser votre site web, il s'agit de vous connecter au site web en tant qu'administrateur.

Pour cela, il suffit de cliquer sur "Admin. du site" en bas de la colonne latérale du site web. Vous pouvez sinon ajouter "wp-admin/" à l'url affichée dans votre navigateur web.

Vous pourrez alors vous connecter avec le nom d'utilisateur et le mot de passe entrés lors de l'installation de votre site internet.

Méthode : Ajouter un article

Dans un premier temps, nous allons enrichir notre site web d'un peu de contenu. Pour cela, nous allons ajouter un article (blog post).

Nous allons remplir cet article avec un nom et du contenu textuel pertinent.

Une fois l'article publié, celui-ci apparait sur votre page d'accueil.

Remarque : Pages

Rendez-vous dans le menu d'administration "Pages". Vous devriez trouver une "Page d'exemple". Vous pouvez regarder son contenu en cliquant sur modifier.

Retournez sur votre site web, êtes vous capable de trouver cette page ?

Méthode : Ajouter un menu

Pour permettre l'affichage des pages, il est nécessaire de créer un menu, puis de sélectionner les pages qui constituent ce menu, et enfin d'insérer ce menu dans notre site web.

Nous allons tout de suite réaliser ces opérations en quelques clics.

Rendez-vous tout d'abord dans "Apparence -> Menu"

Cliquer sur créer un nouveau menu. Dans notre cas, nous avons appelé notre menu "Les matières" puisqu'il sera constitué de pages décrivant chacune des matières présentes à l'IUT.

Sélectionnez ensuite la "Page d'exemple" et cliquez sur "Ajouter au menu".

La page apparaît alors dans le menu comme ci-dessous.

Enfin, pour que le menu apparaisse, il nous faut dire où nous allons placer le menu. Pour cela, cliquer sur "Gérer les emplacements".

Puis choisissez de mettre le menu en haut.

Si vous revenez sur votre site web, vous verrez alors le nouveau menu qui apparaît.

Il suffit ensuite d'ajouter l'adresse du lien puis de la valider. Le texte mis en surbrillance devient alors bleu.

Cette barre ci-dessous nous permet donc de modifier notre texte. Le "B" sert à mettre le texte en gras, "I" en italique.

Les trois icônes suivantes servent respectivement à aligner le texte : à gauche, au centre, à droite.

Méthode : Ajouter des images

Pour vous constituer une banque d'images utilisables dans votre site internet, vous pouvez utiliser l'onglet d'administration décrit ci-dessous, la bibliothèque media. Une fois téléchargée dans la bibliothèque média, les images sont réutilisables partout, dans vos pages et dans vos articles.

Méthode : Personnaliser votre site web

Bon, on sait créer du contenu, rendre visible de contenu à travers les menus, personnaliser ce contenu grâce à l'éditeur de texte, y ajouter des images... Maintenant, reste plus qu'à personnaliser le visuel du site web pour qu'il vous ressemble. C'est pas la partie la plus facile, surtout pour les daltoniens, mais c'est ce qui vous démarquera des autres sites web développés avec wordpress.

Il s'agit de cliquer dans le menu "Apparence -> Personnaliser" comme ci-dessous.

Ensuite, vous accédez au menu qui va vous permettre de personnaliser votre site web :

  • Le thème, son aspect visuel global ;

  • L'identité du site : son titre, logo ;

  • Les couleurs utilisées ;

  • La bannière ;

  • Les menus, comme nous l'avons déjà fait ;

  • Les widgets qui permettent de personnaliser la colonne de droite et le footer.

L'identité du site nous permet donc de personnaliser le titre, le logo et la phrase d'accroche du site web affichée en dessous du titre.

Les menus sont personnalisables comme nous l'avions déjà vu.

Les widgets nous permettent de personnaliser la colonne de droite et le bas de page avec :

  • une barre de recherche ;

  • les commentaires laissés sur les articles ;

  • les archives du site organisés par mois ;

  • les articles organisés par catégorie.

Nous pouvons également ajouter des widgets comme des calendriers ou des images.

La liste des widgets est visible ci-dessous.
